# CHIC
骨格診断とパーソナルカラー診断を行い、それらに基づいて最適なファッションコーディネートを提案します。ユーザーの骨格やパーソナルカラーに合わせ、具体的なスタイルを提案することで、新しい服選びをサポートします。
